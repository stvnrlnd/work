# houseplans.net — Overview

## What It Is

houseplans.net is the primary e-commerce platform operated by [[20_areas/abhp|America's Best House Plans (ABHP)]]. It sells house plans directly to consumers. Steven works on this codebase as part of his day job.

## Tech Stack

Procedural PHP backend (with some Object-Oriented PHP from small modernization efforts). Steven's primary day-to-day work involves feature development, bug fixes, and platform maintenance across the site.

## Development Environment

Unlike [[20_areas/segment-holdings|Segment Holdings]] projects (which run locally via Laravel Sail or Laravel Herd), houseplans.net runs on a **remote development server accessed via SSH**. There is no local development environment.
