# openfaith.world — Changelog

## 2026-04-15 (Decisions)

- Clarified project identity: not faith-oriented, but a community hub rising above religion — uniting people in kindness, love, compassion, and understanding
- Documented tech stack: Statamic (statically generated), Cloudflare Pages, nightly GitHub Action for scheduled post publishing
- Defined monetization direction:
  - Single newsletter with free + premium content in every issue
  - Pay-what-you-want model for premium newsletter tier
  - Annual content packs (fixed price) — 52 or 104 sermon-style pieces + bonus evergreen materials (bookmarks, brochures, etc.)
  - Old content packs feed the premium newsletter tier — natural content reuse pipeline
  - Sponsors receive premium newsletter content, content pack discount, and/or a site profile with sponsor badge
