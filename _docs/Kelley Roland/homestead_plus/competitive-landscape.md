# homestead.plus — Competitive Landscape

*This is a working analysis based on observation of the current market. Update as research continues.*

---

## Primary Competitor: America's Best House Plans (houseplans.net / ABHP)

The closest reference point for homestead.plus. The largest and most established floor plan marketplace in the US.

### What they do

- Large catalog of floor plans from many designers
- Searchable by style, size, bedrooms, bathrooms, garage, stories, and more
- Multiple purchase tiers per plan: Study Set, PDF Set, CAD Files, Reproducible Masters, Multi-Build License
- Plans are credited to "Designers" (ABHP's terminology) — the same convention homestead.plus adopts

### What's broken or missing

- **File store, not a platform** — the experience ends at download. There is no ongoing relationship between the customer and the platform after purchase. No viewer, no project management, no build team.
- **No variant model** — a 2BR and a 3BR version of the same design are listed as completely separate, unrelated products. There is no structural connection between them.
- **No professional network** — ABHP does not connect customers to builders, contractors, or other professionals. The plan sale is transactional and isolated.
- **No viewer** — customers receive a PDF or file package. No in-browser viewing; no IP protection beyond license terms.
- **Flat catalog** — no meaningful "after purchase" layer. Once you download, the platform has no further role in your build journey.
- **Dated UX** — the product and purchase experience has not evolved significantly in years. Feels like a 2010s e-commerce site.

---

## Other Players

### The Plan Collection (theplancollection.com)

Similar model to ABHP — large catalog, PDF/CAD purchase tiers, designer attribution. Search and filtering are slightly more polished. Same structural weaknesses: file store model, no viewer, no post-purchase platform layer.

### Dream Home Source (dreamhomesource.com)

Aggregator-style site pulling plans from multiple designers. Less curated. Same model.

### Family Home Plans (familyhomeplans.com)

Smaller catalog. Similar tiered pricing. No differentiation on the delivery or platform side.

### General Observations Across All Competitors

- All are fundamentally file stores with search
- None have a viewer
- None have project management
- None connect customers to a broader professional network
- None have a concept of plan variants — related configurations are separate unrelated listings
- None offer services (plan modifications, custom design) in an integrated way
- Business/designer onboarding is usually manual and opaque

## homestead.plus — Differentiators

| Feature | ABHP / Competitors | homestead.plus |
|---|---|---|
| File delivery | Download only | Viewer-first; download is designer-controlled |
| IP protection | License terms only | Viewer with browser-level download prevention |
| Plan variants | Separate unrelated listings | Variants on a single product (2BR / 3BR, etc.) |
| Post-purchase | Nothing | Order history, re-view anytime, add-ons |
| Project management | None | Projects with addresses, floor plan assignments, members |
| Build team | None | Curated roster of professionals per project |
| Professional network | None | Directory with preferred partner connections |
| Services | None | Modifications, custom design, made-to-order reports |
| Business tools | Minimal | Full app panel: product management, orders, profile |
| Discovery | Search only | Search + connections + preferred partner referrals |

---

## Positioning Summary

The existing floor plan market is a collection of file stores. Customers buy a plan and are immediately on their own. homestead.plus treats the floor plan purchase as the *beginning* of the build journey, not the end — connecting it to a project, a team of professionals, and an ongoing relationship with the designer and platform.

The viewer-first delivery model is the sharpest single differentiator: it signals that homestead.plus is a platform, not a download site, and it addresses the IP concerns that keep quality designers from listing on existing marketplaces.
