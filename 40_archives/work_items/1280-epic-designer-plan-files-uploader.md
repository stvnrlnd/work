# 1280-epic-designer-plan-files-uploader

## Description

Note for Taylor: This one was already estimated but just needs to be broken up into an epic.

Ability to drop plan files into the portal instead of sending an email to us for new plans. This uploader would need to populate the request in the Trello Publishing board, so we probably want the uploader to send a notification email to [publishing@houseplans.net](mailto:publishing@houseplans.net) after it completes, with a link to the files and the designer information attached.

It would be convenient to upload these files directly to google drive rather than save them on our server.

---

- Uploads should land in google docs ‘Designer Uploads’ directory. Internal directories should be labeled with designer prefix and name, then organized by the designer plan name.
    
- Collect the designer’s plan name and a note field along with the files being uploaded.
    
- Show designers a log of plans and associated upload history. The table should show the abhp plan number in addition to the designer plan name whenever it is available.
    
- Allow additional file uploads against an existing plan
