---
name: Daily note brief detail and linking
description: Daily note brief should match the detail level of the in-chat brief, and use wiki links for projects/areas/notes
type: feedback
---

When writing the ## Brief section of a daily note, the content should match the full detail level of the brief delivered in chat — including all sections (In flight, Blocked/needs a decision, Top 3 priorities) with the same depth and reasoning. Do not write a condensed version.

Also use Obsidian wiki links ([[project-name]]) when referencing projects, areas, or other linkable notes in the daily note.
