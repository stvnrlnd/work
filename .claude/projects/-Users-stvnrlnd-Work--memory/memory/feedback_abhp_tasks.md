ABHP (America's Best House Plans / houseplans.net) tasks are Steven's day job tasks copied from Trello. They exist only as local markdown files in 40_archives/tasks/ for vault context and morning brief reference.

NEVER create a Dart task for an ABHP task. Dart is exclusively for Segment Holdings projects.

ABHP tasks use the template at 30_resources/templates/task.md with source: abhp. The `created` field represents when the task was made ready for development, not when it was added to Trello.
