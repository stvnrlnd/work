ABHP (America's Best House Plans / houseplans.net) work items are Steven's day job tasks copied from Trello. They exist only as local markdown files in 40_archives/work_items/ for vault context and morning brief reference.

NEVER create a Dart task for an ABHP work item. Dart is exclusively for Segment Holdings projects.

ABHP work items use the template at 30_resources/templates/work-item.md. The `created` field represents when the work item was made ready for development, not when it was added to Trello. Filenames use the Trello slug format (e.g. 2695-default-elevation-check.md).
