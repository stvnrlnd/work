description: Always link projects, areas, tasks, and any other referenceable vault notes using [[wiki links]]

When writing or editing any vault file — including daily notes, briefs, changelogs, meeting notes, inbox captures, and system files — always use Obsidian wiki links ([[filename]]) when referencing:
- Projects (e.g. [[homestead.plus]], [[sendfeed.to]])
- Areas (e.g. [[kelley-roland]], [[allgood-roland]])
- Tasks (e.g. [[2343 - Plan Media Viewer - Milestone 3]])
- Any other note that exists in the vault

Do not mention these by name in prose without linking them.
